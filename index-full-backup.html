Setup

Step 1: Clone and Install

First, clone the repository and navigate to the project directory.

$ git clone https://github.com/Jeanrooy/nvidia-claude-proxy.git
Cloning into 'nvidia-claude-proxy'...
$ cd nvidia-claude-proxy
$ pip install -r requirements.txt
Installing collected packages...

This installs FastAPI, httpx, and other dependencies needed to run the proxy server.

Project Structure
python-proxy/ ├── main.py # FastAPI app ├── config.py # .env loader ├── models.py # Pydantic models ├── translation.py # API translation ├── requirements.txt # Dependencies ├── .env # Your config └── .env.example # Template
Setup

Step 2: Configure .env

Copy the example file and fill in your NVIDIA API key and other settings.

$ cp .env.example .env
Now edit .env with your values:
# Address to listen on ADDR=127.0.0.1:8089 # NVIDIA NIM endpoint UPSTREAM_URL=https://integrate.api.nvidia.com/v1/chat/completions # Your NVIDIA API key PROVIDER_API_KEY=nvapi-your-key-here # Optional: protect with auth SERVER_API_KEY=
Usage

Step 3: Run and Connect

Start the proxy server, then configure Claude Code to use it.

$ python main.py
INFO: Started server process
INFO: Waiting for application startup.
INFO: Uvicorn running on http://127.0.0.1:8089
Configure Claude Code

Create .claude/settings.json in your project:

{ "env": { "ANTHROPIC_BASE_URL": "http://localhost:8089", "ANTHROPIC_API_KEY": "YOUR_NVIDIA_API_KEY", "model": "z-ai/glm4.7" } }
